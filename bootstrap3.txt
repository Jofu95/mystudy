bootstrap是基于html、css、js
用于快速开发web应用程序和网站的前端框架
移动设备优先、浏览器支持、响应式设计

内容：
	基本结构：带有网格系统、链接样式、背景的基本结构
	css：全局css设置、定义基本的html元素样式、可扩展的class、先进的网格系统
	组件：创建图像、下拉菜单、导航、警告框、弹出框等
	js插件：自定义jQuery插件
	定制：

在引入bootstrap的js文件前需要先引入jquery的js文件

Bootstrap CSS
	html5文档类型：
		<!DOCTYPE html>
		<html>
			<head>
				<meta charset='utf-8'/>
			</head>
		</html>
	viewport meta 标签：
		<meta name='viewport' content='width=device-width,initial-scale=1.0,user-scalable=no'/>
		width:控制设备的宽度，确保在不同设备正确呈现
		initial-scale=1.0 确保网页加载以1:1比例呈现，不会有任何缩放
		user-scalable=no 禁止其在移动设备上缩放功能
	响应式图像：
		添加img-responsive class让图片支持响应式布局
		.img-responsive{max-width:100%;height:auto;display:inline-block}
	全局布局、排版、链接
		基本全局显示：移除body边距、设置默认字体大小、字体样式、行高、文本颜色、背景色
		排版：@font-family-base、@font-size-base、@line-height-base
		链接样式：@link-color设置全局颜色
	容器（container）：
		container class用于包裹页面上的内容
		
	网格系统：
		移动设备优先策略：内容(决定什么是最重要的)、布局(优先设计更小的宽度)、渐进增强(随屏幕大小增加而添加元素)
		响应式网格系统随屏幕或视口尺寸的增加，系统会自动分为最多12列
		工作原理：
			1.行必须放在.container class内
			2.行创建水平组
			3.内容应该放在列内
			4.列可通过内边距来创建内容之间的间隙
		媒体查询：
			超小设备(手机<768px) 		Bootstrap默认情况下没有媒体查询
			小型设备(平板>=768px)		@media (min-width: @screen-sm-min){...}
			中型设备(电脑>=992px)		@media (min-width: @screen-md-min){...}
			大型设备(大电脑>=1200px)		@media (min-width: @screen-lg-min){...}
			
	排版：
		强调：
			text-left|center|right class 左中右对其
			text-primary|success|info|warning|danger	不同颜色的字体
		
	表格：
		条纹表格：添加 .table-striped class （在<table>标签上）
		边框表格：添加 .table-bordered class （在<table>标签上）
		悬停表格：添加 .table-hover class （在<table>标签上）
		精简表格：添加 .table-condensed class （在<table>标签上）
		响应式表格：添加 .table-resonsive class （在<table>标签上）
		上下文表格：添加 .active|.success|.warning|.danger class （在<tr>、<td> 或 <th>添加）
		
	表单：
		表单布局：
			垂直表单（默认）：
				<form>标签添加role="form",标签和控件放在带有.form-group class的div内,文本标签添加.form-control class
			内联表单：
				左对齐，标签并排，<form>标签添加.form=inline class
			水平标签：
				<form>标签添加.form=horizontal class,标签和控件放在带有.form-group的div内，标签添加.control-label
		表单控件状态：
			禁用输入框	disabled
			禁用字段集	对<fieldset>标签添加 disabled属性来禁用标签内所有控件
			验证状态	.has-warning、.has-success、.has-error

	按钮：
		按钮样式：
			.btn-default（默认）|.btn-primary|.btn-success|.btn-info|.btn-warning|.btn-danger|.btn-link
		按钮大小：
			.btn-lg|.btn-sm|.btn-xs|.btn-block
	
	图像：
		.img-rounded		添加border-radius:6px来获得图像圆角
		.img-circle			添加border-radius:500px来让整个图像变成圆形
		.img-thumbnail		添加一些内边距和一个灰色边框
